# Test Conditions To Pseudocode

Use the bundled Python script to read a JSON file generated by the test-item extraction workflow and convert each test item into exchange-style pseudocode JSON.
Prefer this skill when the input already contains structured fields such as `test_name`, `test_method`, `unit`, and `pins`, and the goal is to transform them into concise executable-style pseudocode.

## Workflow

1. Confirm the input JSON path. The input should come from a test-plan extraction step such as `excel-read-test-items`.
2. Read `references/exchange-rules.md` when you need the pseudocode vocabulary semantics.
3. Convert `pins` into setup commands such as `FV`, `FI`, `Float`, `MV`, or `MI`.
4. Convert `test_method` natural language into additional pseudocode commands when clear patterns are recognized.
5. Emit JSON in the same high-level shape as `FT_Test_Plan_JSON.json`, with:
   - `pin_order`
   - `records`

## Commands

Default conversion:

```powershell
python scripts/convert_test_conditions.py --input FT_Test_Plan_JWXX-JPXX-Final-test-plan_test_items.json --pretty
```

Write to a specific file:

```powershell
python scripts/convert_test_conditions.py --input FT_Test_Plan_JWXX-JPXX-Final-test-plan_test_items.json --output FT_Test_Plan_pseudocode.json --pretty
```

## Output Rules

- Support `.json` input only.
- Expect input generated from the test-item extraction workflow.
- Preserve `pin_order` in the output.
- Output `records` as compact arrays in the style:
  - `[section, item_id, test_name, pseudocode, unit]`
- Use exchange-style commands such as `FV`, `FI`, `Float`, `RampV`, `RampI`, `MV`, `MI`, `MT`, `Connect`, `Display`, `Delay`, and `WR`.
- Prefer deterministic conversions. If a natural-language phrase is not confidently recognized, keep the converted result as concise as possible and avoid inventing unsafe semantics.

## Files

- `scripts/convert_test_conditions.py`: main JSON-to-pseudocode converter
- `references/exchange-rules.md`: pseudocode vocabulary reference
- `agents/openai.yaml`: skill UI metadata
